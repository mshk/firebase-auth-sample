Sample 'Firebase Auth' project with React.js featuring:
* Email/Password Login
* Twitter Login

# How to run

```
$ git clone https://github.com/mshk/firebase-auth-sample
$ cd firebase-auth-sample
$ yarn
$ yarn run start
$ open http://localhost:3000/ 
```
